Active Directory Users & Computers

Active Directory Users & Computers (ADUC) is a industry renowned tool utilized by admins to perform daily tasks and much more in Active Directory. ADUC also provides an object based view of the domain in the Active Directory environment. Users can be added to a group and and will have corresponding access.
