Active Directory Automation with Powershell:

Active Directory automation can be used be performed by different scripting languages. The one we will assess is powershell. Powershell is a powerful language to automate repetetive active directory tasks that would have normally been done manually. Manual task completion proves time consuming and automation is the way to go. Powershell is a powerful task automation framework by Microsoft.
